# 💻 Inventory Laptop & Software Management System

“I developed a web-based Laptop and Software Inventory Management System. The objective is to manage company assets like laptops and software in a centralized way.
Previously, companies used Excel sheets which are not efficient for tracking and maintaining consistency.
My system helps in tracking asset allocation, monitoring usage, and generating reports. It also provides forecasting based on company growth to estimate future asset requirements  based on 5% anual employee growth and identify unused software to reduce cost.”

---

## 🚀 Features

### 📦 Laptop Inventory Management
- Manage multiple laptop models  
- Track individual laptop assets using serial numbers  
- Monitor status (Available, Assigned, Under Repair)  

### 🧑‍💼 Employee Management
- Store employee details  
- Track assigned assets per employee  

### 🔄 Asset Assignment System
- Assign laptops/software to employees  
- Prevent duplicate assignments  
- Return and reassign assets  

### 💿 Software License Management
- Manage software models and licenses  
- Track assigned software to employees  

### 📊 Dashboard & Analytics
- View total assets, assigned, and available stats  
- Real-time tracking of inventory usage  

### 🔐 Authentication & Authorization
- JWT-based login system  
- Role-based access (Admin, IT Operations)
